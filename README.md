# 지하철 카운트다운

역과 방향을 선택하면 실시간으로 남은 초를 카운트다운해주는 모바일 웹앱입니다.
서울시 열린데이터광장의 "지하철 실시간 도착정보" API를 사용합니다.

## 1. API 키 발급 (1회만, 무료)

1. https://data.seoul.go.kr 접속 후 회원가입/로그인
2. 우측 상단 "인증키 신청" 또는 검색창에 "지하철 실시간 도착정보" 검색
3. 해당 API 페이지에서 [활용신청] → 즉시 자동 승인되어 인증키 발급
4. 발급된 인증키를 복사

## 2. 설치 및 실행

```bash
cd subway-countdown
npm install
cp .env.example .env
# .env 파일을 열어 SEOUL_API_KEY=발급받은키 로 수정
npm start
```

서버가 뜨면 http://localhost:3000 에서 확인할 수 있습니다.

## 3. 휴대폰에서 사용하기

- **같은 Wi-Fi에서 테스트**: 컴퓨터의 로컬 IP(예: 192.168.0.5)로 `http://192.168.0.5:3000` 접속 후, 홈 화면에 추가하면 앱처럼 사용 가능
- **집 밖(이동통신)에서도 쓰려면**: Vercel, Railway 등 무료 호스팅에 배포해야 합니다. 배포를 원하시면 말씀해주세요 (계정 생성이 필요해 직접 진행해드릴게요).

## 사용 방법

1. 역 이름 입력 (예: 강남) → 검색
2. 방향/행선지 카드 선택 (예: "성수행 - 신도림 방면")
3. 큰 숫자로 도착까지 남은 초가 카운트다운됩니다. 15초마다 실제 데이터로 자동 보정됩니다.
4. ☆ 버튼으로 즐겨찾기 등록하면 다음부터는 검색 없이 홈 화면에서 바로 탭 가능합니다.
